## Summary

Hackweek project to prototype an ML-driven recommendation engine using past search and purchase behavior to cluster users and suggest relevant items.

## Problem

Search results did not fully leverage user behavioral history for personalized recommendations.

## Goal

Analyze user activity, cluster behavior, and generate recommendations based on cluster similarity.

## Responsibilities

- Designed the recommendation concept, built prototype clustering logic, and presented at Hackweek.

## Technical concept

1. Collect user activity data
2. Analyze search and purchase history
3. Tag behavioral attributes
4. Cluster users and generate recommendations

## Impact

- Prototype received recognition and later evolved into production features.
