## Summary

Hackweek project that evolved into an internal UI component system to standardize design, improve developer productivity, and ensure accessibility.

## Problem

UI components were inconsistent across teams, causing variable experiences and reduced developer productivity.

## Goal

Create a shared component system (buttons, inputs, forms, layout) with accessibility and consistent styling.

## Role

- Co-created the idea during Hackweek, collaborated with a Senior Staff engineer, conducted cross-team research, and implemented prototype components.

## Impact

- Platform later adopted across products, open-sourced as part of eBay Skin, and reduced accessibility bugs.

## Recognition

- Received a Spot Award for the contribution.
